# Yasir Azam | Cinematic Scrollytelling Portfolio

A high-end, immersive personal portfolio built with **Next.js 14**, **TypeScript**, **Framer Motion**, and **Tailwind CSS**. This project features a custom frame-synced scrollytelling engine, cinematic UI effects, and a live production-grade contact backend.

## 🚀 Key Features

### **1. Cinematic Scrollytelling Engine**
- **Hardware-Locked Canvas:** Uses HTML5 Canvas to render a 75-frame image sequence.
- **Frame-Locked Logic:** Overlay text is mapped to specific frames for 100% reliable visibility.
- **Smooth Scroll:** Powered by **Lenis** for ultra-fluid navigation.

### **2. Elite UI Effects**
- **Typewriter Reveal:** Staggered typing effect for all headers.
- **3D Card Tilt:** Achievement cards physically tilt toward the cursor using spring physics.
- **Magnetic Buttons:** Social links attract the cursor locally.
- **Filmic Grain Overlay:** Subtle, animated noise texture for a premium look.
- **Custom Cursor:** Advanced physics-based cursor that reacts to interactive nodes.

### **3. Production Backend**
- **Live Form Integration:** Connected to **Web3Forms API**.
- **SEO Optimized:** Complete with `sitemap.ts`, `robots.ts`, and optimized metadata.

## 🏗️ Getting Started

### **Prerequisites**
- Node.js 18.x or higher

### **Installation**

1. **Clone & Install:**
   ```bash
   npm install
   ```

2. **Configure Assets:**
   - Place image sequence (0-74) in `public/sequence/`.
   - Place CV as `Yasir_Azam_CV.pdf` in `public/`.

3. **Run Dev Server:**
   ```bash
   npm run dev
   ```

---

## 📂 Structure
- `src/app/`: Root layouts and SEO.
- `src/components/`: Core logic (ScrollyCanvas, TiltCard, Magnetic, Typewriter).
- `public/`: Static CV and Image assets.
